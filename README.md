# 📝 Modern To-Do List App

A beautiful, modern, and user-friendly **To-Do List** application built with **React**, **Material-UI (MUI)**, and **Vite**.  
Designed for productivity, premium aesthetics, and seamless task management—ideal for individuals who value both function and form.

---

## ✨ Features

- **🖥️ Modern, Responsive UI**  
  Sleek, minimalist design with white-to-gradient background, smooth transitions, and full mobile responsiveness.

- **➕ Add, ✏️ Edit & 🗑️ Delete Tasks**  
  Easily manage your to-dos with intuitive controls, including a standout yellow "+" button for quick additions.

- **📋 Detailed Tasks**  
  Add a title, description, due date, category, and priority level to every task for better organization.

- **✅ Bulk Actions**  
  Mark all tasks as complete or clear all completed tasks with a single click.

- **🔍 Smart Search & Filter**  
  Instantly search and filter by **category** or **priority** to find what matters fast.

- **🌙 Dark & Light Mode**  
  Toggle between light and dark themes for a personalized viewing experience.

- **💾 Persistent Storage**  
  Your data is saved in `localStorage`, ensuring your tasks persist between sessions.

- **📱 Accessible & Mobile-Friendly**  
  Optimized layout and large touch targets make it easy to use across all devices.

## 🚀 Getting Started

Follow these steps to run the app locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
````

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the development server:**

   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Visit [http://localhost:5173](http://localhost:5173) or the port shown in your terminal.
